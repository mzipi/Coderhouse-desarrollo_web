Desarrollo web

El concepto de desarrollo web

El desarrollo web es el proceso que se lleva a cabo para crear un sitio web. El desarrollador codifica la web con lenguajes como HTML, CSS y JavaScript. Estos archivos que a simple vista son solo texto, son interpretados por un navegador web para renderizar en la pantalla lo que el código le especifica.

Desarrollador web vs diseñador web

El desarrollador web es la persona que se encarga de que la funcionalidad del sitio web sea el correcto.
El diseñador web por otra parte es la persona que tiene los conocimientos necesarios para que su sitio tenga el aspecto deseado.
Por supuesto es posible que un profesional tenga conocimientos de desarrollo web y de diseño web.
