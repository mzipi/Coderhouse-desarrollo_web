Node

Node es un framework basado en el motor de javascript de Google llamado V8.
Esto nos permite crear aplicaciones web de manera rápida y sencilla.
Node además, mediante npm, es capaz de instalar paquetes de terceros que amplían la funcionalidad de nuestras aplicaciones.

Boostrap

Boostrap es un biblioteca que los desarrolladores web utilizan para dar estilos facilmente a sus páginas web.
También es posible personalizar los estilos pre-establecidos, e incluso crear una plantilla de página web.

Sass

Sass es una herramienta utilizada por los desarrolladores web que potencian las caracteristicas que proporcinan las hojas de estilo en cascada.
Esta herramienta nos permite utilizar variables, realizar operaciones, combinar y reutilizar codigo.
