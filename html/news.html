React 18

React 18 incluirá mejoras listas para usar (como procesamiento por lotes automático), nuevos APIs (como startTransition), y una nueva transmisión para el renderizado en el servidor con soporte incorporado para React.lazy.

Angular 13

El motor View Engine ya no estará disponible. El APF (Formato de Paquete Angular) se ha simplificado y modernizado. La API de componentes se ha modificado. IE 11 deja de tener soporte. Mejoras en Angular CLI.
